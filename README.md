MaghSec is a Morocco-based cybersecurity initiative focused on **responsible vulnerability disclosure** and **security research** across the Maghreb and North Africa region.
The platform connects organizations with skilled security researchers to help identify and remediate security vulnerabilities before they can be exploited.

# Focus Area
- Morocco
- North Africa / Maghreb region
- [![Live Website](https://img.shields.io/badge/Live%20Site-View-success?style=flat-square&logo=githubpages)](https://monsifhmouri.github.io/MaghSec/)


# What MaghSec Provides
Responsible Vulnerability Disclosure framework
Clear rules and legal boundaries for security testing
Secure communication between companies and researchers
Ethical and lawful security research practices

# Important Notice
This repository hosts the **public website frontend only**.  
It does **not** contain backend logic, sensitive data, or internal security mechanisms.
# Legal & Ethics
All security testing must be:
Explicitly authorized
Within defined scope
Conducted under responsible disclosure principles
Compliant with applicable laws and regulations
